# 📄 Descargar PDF de Google Drive Bloqueados

Herramienta simple y rápida para **descargar PDFs de Google Drive** que están bloqueados por la consola de desarrollador (vista previa restringida).

### 🚀 Cómo usar (muy fácil)

1. **Descarga el archivo**  
   [CodeDownloadPDF.js](CodeDownloadPDF.js) ← haz clic aquí para descargarlo

2. Abre el PDF bloqueado en Google Drive (en modo vista previa)

3. Presiona **`F12`** en tu teclado para abrir las **Herramientas de desarrollador**

4. Ve a la pestaña **Consola**

5. Copia **todo** el contenido del archivo `CodeDownloadPDF.js`

6. Pégalo en la consola y presiona **Enter**

   > Si te sale el mensaje de error:  
   > **"Paste is blocked..."**  
   > Solo escribe esto en la consola y presiona Enter:  
   > `allow pasting`  
   > Luego pega el código nuevamente y presiona Enter.

7. El PDF se descargará automáticamente como **`download.pdf`** con:
   - Tamaño original de la imagen
   - Sin bordes blancos
   - Sin distorsión

¡Listo! ✅

### Características

- Usa la resolución real de las imágenes (máxima calidad)
- Mantiene la proporción original (no estira ni distorsiona)
- Cada página del PDF tiene exactamente el tamaño del PDF original
- Funciona con la versión antigua de jsPDF (la más compatible)
- No requiere instalar nada

### Notas importantes

- Solo funciona mientras tengas el PDF abierto en la vista previa de Google Drive.
- Funciona en Chrome, Edge y la mayoría de navegadores de escritorio.
- No guarda ni envía tus archivos a ningún lado (todo se ejecuta en tu navegador).
